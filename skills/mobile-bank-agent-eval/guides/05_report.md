# Guide 05 — 报告生成

HTML + Markdown 双格式报告。

## HTML 报告内容

1. **汇总卡片**: 用例总数/通过/失败/阻塞/通过率
2. **进度条**: 可视化 pass/fail/blocked 比例
3. **维度分析**: 按维度统计 + 用例表
4. **失败归因**: F1-F8 分类
5. **Trace 调用结构**: 树状图展示每 case 的调用链
6. **响应时间分析**: avg/p50/max

## 失败归因

| 类型 | 说明 |
|------|------|
| F2.1 | 任务理解失败 |
| F5.3 | 服务异常 |
| F7.1 | 输出格式不符 |
| F7.3 | 输出缺关键内容 |
| F8.1 | 执行超时 |

## 输出

- test_report.html: 专业 HTML（内联 CSS，可邮件分享）
- test_report.md: Markdown 版本
