# 测试执行记忆库模板

这是一个通用的测试执行记忆库结构模板。当新建一个测试项目工作区时，将此模板复制到工作区的 `.memory/` 目录，并根据项目实际情况填充内容。

## 使用方式

1. 将本模板复制到工作区：`.memory/`
2. 根据项目实际情况修改模板中的占位符
3. 删除 `.example-module/` 目录，创建实际模块目录
4. 按需要新增 shared 文件
5. 新增案例时，复制 `cases/case-template.md` 并填充内容

## 设计原则

- 项目级记忆按业务模块组织
- 通用规则按操作对象组织到 test-web-execution skill
- 历史案例按一次执行一个文件保存
- 每个文件控制在 50-100 行
