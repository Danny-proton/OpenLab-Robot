# OpenLab Robot

#### 介绍
Openlab-Robot
把网页录制回放、智能执行（BrowserUse）、桌面智能执行（ComputerUse）、会话、多项目、分支 / Worktree、右侧代码改动、代码 Diff、权限审批、模型提供商、Computer Use、H5 远程访问、welink接入和定时任务集中到一个Windows APP 里。支持多AI模型集成、技能系统、插件框架、渠道管理、记忆系统等企业级功能。
